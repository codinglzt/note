```
function binarySearch(arr, num) {
  let l = 0, r = arr.length - 1;
  while (l <= r) {
    let middle = Math.floor(l + (r - l) / 2);
    if (arr[middle] === num) {
      return middle;
    } else (arr[middle] > num) {
      r = middle - 1;
    } else {
      l = middle + 1;
    }
  }

  return -1;
}
```

### 时间复杂度
  二分法每次能将查找区间减半，时间复杂度为O(logN)

### 注意
  middle 值的计算： middle = l + (r - l) / 2, l + r 时可能会出现加法溢出的情况。
